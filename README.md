# Digital Literacy Project

## Description

This project focuses on understanding and applying the concept of digital literacy through a series of practical tasks. It covers important areas such as online safety, responsible use of digital tools, professional communication, and awareness of cyber risks. The aim of this project is to develop a better understanding of how to use digital technologies effectively and responsibly in everyday life.


## Tasks Covered

### Task 1 — Infographic
Created a digital literacy infographic using Canva. It highlights key concepts such as safe internet practices, useful digital tools, professional online presence and email etiquette.

### Task 2 — Digital Portfolio
Developed and reviewed digital profiles on GitHub and LinkedIn to build a professional online presence and showcase projects.

### Task 3 — Platforms
Solved a beginner-level problem on HackerRank to practice coding skills. Also created a Google Form titled “Digital Literacy Awareness Quiz” to understand how online tools can be used for surveys and data collection.

### Task 4 — Email Etiquette
Wrote two professional emails (assignment extension and internship request) and created a social media checklist covering safe and responsible online behaviour.

### Task 5 — Cybercrime Case Study
Analyzed a case study related to digital literacy and online safety, focusing on ethical behaviour, digital identity and responsible use of technology.

Smart Password Strength Checker


Overview

The Smart Password Strength Checker is a Python-based application that evaluates the strength of a user’s password. It analyzes different characteristics such as length, use of uppercase and lowercase letters, numbers and special characters. Based on these factors, the system classifies the password as weak, medium or strong. This project focuses on enhancing cyber awareness and digital literacy by helping users understand how secure their passwords are and encouraging better password practices.


Features

- Checks password strength in real-time

- Evaluates multiple security parameters:

- Minimum length

- Uppercase and lowercase characters

- Numeric digits

- Special symbols

Classifies passwords as:

- Weak

- Medium

- Strong

Simple and easy to run from the terminal.


Each password is analyzed based on different criteria and a score is assigned. The final classification is determined using this score.

Technologies Used

- Python 3

- Regular Expressions (re module)
