# Digital Literacy Project 

### Description  
This project explores the idea of digital literacy through practical tasks. It includes important topics like online safety, responsible use of digital tools, professional communication, and understanding cyber risks. The goal of this project is to help people learn how to use digital technologies effectively and responsibly in their daily lives.

### Smart Password Strength Checker  
## Overview  
The Smart Password Strength Checker is a Python application that evaluates the strength of a user’s password. It analyzes various characteristics like length, use of uppercase and lowercase letters, numbers, and special characters. Based on these factors, the system classifies the password as weak, medium, or strong. This project aims to improve cyber awareness and digital literacy by helping users understand password security and encouraging better password practices.  

## Features  
Checks password strength in real-time.  
Evaluates multiple security parameters:  
- Minimum length  
- Uppercase and lowercase characters  
- Numeric digits  
- Special symbols  

Classifies passwords as:  
- Weak  
- Medium  
- Strong  

It is simple to run from the terminal. Each password is analyzed based on different criteria, and a score is assigned. The final classification is based on this score.  

## Technologies Used  
- Python 3  
- Regular Expressions (re module)
